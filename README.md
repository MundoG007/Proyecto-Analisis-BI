# An-lisis-de-Inversiones-y-Negociaci-n-en-Power-BI
Proyecto de análisis en Power BI basado en datos de una empresa. Explora patrones de inversión, relación entre valuación y equity, tasa de acuerdos y comportamiento de los inversionistas. Incluye modelado de datos, transformación en Power Query y métricas en DAX para generar insights estratégicos.
